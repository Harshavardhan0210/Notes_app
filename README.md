# Notes App (Python + HTML + TXT Storage)

## Project Overview
This is a simple Notes Application built using **Python (Flask)** and **HTML**.

The app allows users to create and store notes. Instead of using a database, all notes are saved in a local text file (`notes.txt`), making it lightweight and beginner-friendly.

---

## Features
- Create new notes
- Save notes in a text file
- View saved notes
- Simple web interface
- No database required

---

## Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML
- **Storage:** Text File (`notes.txt`)

---

## How to Run the Project

### Step 1: Clone or Download the Project

### Step 2: Open Project Folder
Open the folder where `app.py` is located.

### Step 3: Open Terminal

### Step 4: Install Flask
```
pip install flask
```

### Step 5: Run the App
```
python app.py
```

### Step 6: Open in Browser
```
http://127.0.0.1:5000/
```

---

## How It Works
- User enters a note in the web page
- Flask backend receives the input
- The note is saved into `notes.txt`
- Notes are displayed on the page

---

## Common Issues
- Make sure Python is installed
- Make sure Flask is installed
- Run using `python app.py`
- Do not open HTML file directly
